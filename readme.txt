== Description ==

Simple CRM is a WordPress plugin that lets you define custom fields to extend user profiles and it is also a framework for integration with all kind of CRM API webservices. BuddyPress addon can export and import XProfile data to it.

Plugin sponsored by [Vivanista.com](http://vivanista.com)
